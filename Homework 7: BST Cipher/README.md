# Homework 7: BST Cipher

## Overview
This assignment implements a custom encryption and decryption algorithm using Binary Search Trees (BST). The program constructs a BST based on a provided cipher key phrase, where each character's unique path through the tree dictates its encrypted code. It then processes standard input to encrypt a plaintext message and decrypt a ciphertext message utilizing this generated BST structure.

## Concepts Covered
* Binary Search Tree (BST) construction and insertion logic
* Custom tree traversals for data encryption and decryption
* ASCII character mapping and comparison
* String manipulation and parsing from standard input
