# Homework 1: Number Spiral (Exceptions)

## Overview
This program generates a mathematical number spiral within a 2D array and computes the sum of adjacent numbers for specified target values. It reads input from a text file, validates the requested spiral dimension (automatically adjusting even sizes to the next odd integer), and uses robust exception handling to gracefully manage invalid data or out-of-bounds inputs.

## Concepts Covered
* 2D array (matrix) initialization and manipulation
* Algorithmic grid traversal and coordinate navigation
* Exception handling (`try`/`except` blocks and raising `ValueError`)
* File I/O operations and strict input validation
