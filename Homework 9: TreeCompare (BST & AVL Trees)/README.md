# Homework 9: TreeCompare (BST & AVL Trees)

## Overview
This assignment implements a versatile `TreeCompare` class that can function as either a standard Binary Search Tree (BST) or a self-balancing AVL Tree, depending on an initialization flag. The program extends a provided tree structure by adding custom methods to calculate the tree's height, determine the mathematical range of its keys, retrieve all nodes at a specific depth level, sum the values of all leaf nodes, and generate a left-side view of the tree.

## Concepts Covered
* Binary Search Tree (BST) and AVL Tree structures
* Conditional self-balancing and tree rotation logic
* Recursive and iterative tree traversals (tracking depth and identifying leaves)
* Object-oriented programming and extending existing class functionality
