# Homework 4: The Josephus Problem (Linked Lists)

## Overview
This assignment solves the classic Josephus Problem by simulating a counting-out game where participants are sequentially removed from a circle until only one remains. It requires building a custom circular linked list from scratch to represent the participants, process the starting positions, and manage the continuous traversal and removal logic. 

## Concepts Covered
* Circular linked list data structure implementation
* Advanced node manipulation and reference management (`first`, `last`, and `next` pointers)
* Algorithmic simulation of continuous cyclic traversal
* Standard input parsing and file I/O operations
