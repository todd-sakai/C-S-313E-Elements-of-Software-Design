# Homework 8: Autocomplete (Tries)

## Overview
This assignment extends a standard Trie (Prefix Tree) data structure to implement a functioning autocomplete engine. The program processes a history of searched words, tracking their individual access frequencies within the tree's nodes. It then takes a series of prefixes, traverses the Trie to generate a list of all matching complete words, and returns the results sorted in descending order based on their historical search frequency.

## Concepts Covered
* Trie (Prefix Tree) data structure implementation and recursive tree traversal
* Object-oriented programming (node manipulation and custom attribute tracking)
* Sorting algorithms and sorting by specific object attributes (frequency)
* Standard input parsing and test case debugging
