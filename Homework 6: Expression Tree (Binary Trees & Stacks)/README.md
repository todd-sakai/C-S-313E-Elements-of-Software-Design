# Homework 6: Expression Tree (Binary Trees & Stacks)

## Overview
This assignment involves implementing a Binary Expression Tree to represent, analyze, and compute algebraic expressions. The program parses a mathematical string character-by-character to build the tree structure, calculates the final mathematical result by recursively traversing the nodes, and outputs both the prefix and postfix notations of the original expression.

## Concepts Covered
* Binary tree data structures (internal operator nodes and leaf operand nodes)
* Stack data structures (utilizing LIFO patterns for expression parsing and tree construction)
* Recursive tree traversal and mathematical evaluation
* Pre-order (prefix) and post-order (postfix) tree traversals
