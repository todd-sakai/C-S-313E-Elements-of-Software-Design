# Homework 2: Working Hard (Binary Search)

## Overview
This assignment simulates a productivity problem to compare the efficiency of linear and binary search algorithms. It features a recursive function to calculate the total lines of code written under exponential productivity loss. It also implements both linear and binary search methods to find the optimal starting lines of code, tracking execution time and recursive function calls to mathematically demonstrate the speed advantage of binary search.

## Concepts Covered
* Binary and linear search algorithm implementation
* Recursive functions and base cases
* Performance profiling (measuring execution time with the `time` module)
* Algorithm efficiency tracking and comparison
