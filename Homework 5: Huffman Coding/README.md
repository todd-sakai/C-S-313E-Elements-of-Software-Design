# Homework 5: Huffman Coding

## Overview
This assignment implements text compression and decompression using the Huffman Coding algorithm. It generates a custom Huffman Tree and character codes based on a predefined alphanumeric message key. The program then reads multiple lines of text from standard input, cleans the data, and processes each message to display its original text, its compressed binary string, and its successfully decompressed output.

## Concepts Covered
* Data compression techniques and Huffman Coding
* Tree data structures and Priority Queues
* Standard input parsing (`sys.stdin`) and string manipulation
* Object-oriented programming (utilizing provided class methods for compression/decompression)
