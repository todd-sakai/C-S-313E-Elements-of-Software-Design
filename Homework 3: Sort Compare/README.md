# Homework 3: Sort Compare

## Overview
This assignment instruments and evaluates the performance of three sorting algorithms: Selection Sort, Insertion Sort, and Quicksort. It involves modifying the algorithms to count the exact number of element comparisons and swaps they execute during runtime. The algorithms are then benchmarked against different types of datasets (small, large, pre-sorted, and reverse-sorted) to empirically analyze their efficiency and time complexity under various data conditions.

## Concepts Covered
* Sorting algorithm implementation (Selection Sort, Insertion Sort, and Quicksort)
* Empirical algorithm analysis and performance profiling
* Tracking specific execution metrics (comparisons and element swaps)
* Evaluating algorithm behavior and efficiency across varying dataset conditions
